# Public Beta Hub

## Purpose

Collect the public beta docs that matter for trying Clawback and understanding
the current `0.4` scope.

This folder is for:

- public beta scope
- public beta operator guidance
- current milestone definition and signoff
- current bounded public-ship status and closeout history
- limitations and verification guidance

It is not the full product or architecture archive.

## What This Folder Should Answer

- what the current beta actually covers
- how a stranger should try the hosted demo first
- what "publicly tryable" means for a self-hosted OSS product
- which setup paths are mandatory versus optional
- what the current beta supports and what it does not
